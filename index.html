My Project Introduction 01

Oct/02/2018

I wanted to design a simple blog home page to present a set of information including my blog, about and contact. I make a navigation for each section on the home page and a ‘home’ button fixed at the bottom of the page to return the home page. On the whole, I try to create a simple, quiet atmosphere and try to make it look beautiful. The target audience of this page is those who want to browse information quickly.

So far, I have read most of the utilities about bootstrap4 at w3school, and I have considered whether to put these features on my personal homepage. For example, I once added a navigation bar to my homepage. I found this feature useful for quickly searching for specific content, but it didn't look like the theme I wanted, so I ended up not using it.

To realize webpage navigation, I adopted button and fixed-bottom functions. I arrange three buttons of 'btn-light' types horizontally as the navigation of the page. I added the 'HOME' hyperlink text and fixed it at the bottom of the page to return to the HOME page at any time. I also added an effect with which the block's background will turn gray when the mouse hovers over it. After I done below, the webpage still looks as clean as I intended.

In the process of coding and learning, when I encounter some knowledge I did not know before, I will check on w3school.com, there are many tutorials for my reference. In the adjustment and modification of the design, I change the code to see the results again and again. Because every time I change the code and run it, there will be timely feedback on the browser so that I can see if the change happened. This kind of timely inspection also helps me to better understand the meaning of the code.

To further develop my project, I plan to add more interactive or animation features. I will pay more attention to these functions in my future study.
